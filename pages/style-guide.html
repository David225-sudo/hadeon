Style Guide

Typography

Hadeon uses a restrained editorial typographic system built around clarity, contrast, and confident spacing.

Primary Typeface

Satoshi is used across the interface for headings, navigation, labels, and long-form supporting copy.

Heading Scale

Large display headings keep a calm weight with tight tracking, while utility labels remain compact and crisp.

Color System

The palette stays monochrome at its core, supported by restrained neutrals and a selective warm accent for emphasis.

Base Colors

Primary surfaces use layered black values, while foreground text balances soft whites and muted grey tones.

Accent Usage

Accent colors are reserved for high-attention moments such as highlights, bars, and interactive emphasis.

Components

Interface Principles

Buttons, cards, menus, and footer elements all follow the same visual rhythm: minimal borders, structured spacing, and subtle motion.

Hover states are clean and intentional, with typography and underline details carrying most of the interaction language.
